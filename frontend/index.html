MKV is recommended for general use — it supports every codec, preserves subtitles, and has no audio restrictions. MP4 offers the widest device playback compatibility but strips subtitles, forces AAC audio, and does not support VP9. WebM is required for Tower Unite and web delivery; it only accepts VP9 or AV1 video with Opus audio — H.264 and HEVC are not valid in WebM.

                Deinterlace
                
Apply the yadif filter to deinterlace interlaced video sources (e.g. old TV captures, DVD rips). Has no effect on already-progressive content. Not applied to remux jobs.

              

              
            

          

          

            

              

                Auto-Crop Black Bars
                
Automatically detect and crop black bars before encoding using a cropdetect pre-pass. Useful for content with letterboxing or pillarboxing. Adds a brief analysis step per file. Not applied to remux jobs.

              

              
            

          

          

            

              

                Denoise
                
Apply the hqdn3d spatial/temporal denoising filter. Useful for noisy sources (old VHS rips, compressed web video). Also helps the encoder achieve better compression. Not applied to remux jobs.

Lossless tracks (TrueHD, DTS-HD MA, FLAC, PCM) are always copied without re-encoding regardless of this setting. If no matching language track is found, the first audio track is kept as a fallback.

          

          

            
            

              
              

                
                
                
                
                
                
                
                
              

            

            
Audio and subtitle tracks for these languages are kept in the output. Tracks in other languages are dropped. Enable "Keep all languages" to include every track regardless of language. If no matching track is found, the first audio track is always kept as a fallback.

          

          

            

              

                Force Stereo Downmix
                
Downmix all audio tracks to stereo (2.0) during re-encoding. Useful for devices that don't support surround sound, or for Tower Unite which only uses stereo. Only affects tracks that are being re-encoded — lossless and copied tracks are unaffected.

              

              
            

          

          

            

              

                Loudness Normalization
                
Apply EBU R128 loudness normalization to re-encoded audio tracks. Targets −23 LUFS with a true peak of −2 dBTP. Useful for libraries with inconsistent volume levels across different sources. Only affects tracks that are being re-encoded.

              

              
            

          

          

            

              

                Force Encode Audio
                
Always re-encode audio to the selected codec, even if the source track is lossless (FLAC, TrueHD, DTS-HD MA). By default, lossless tracks are copied as-is to preserve quality. Enable this to ensure all audio is encoded to Opus or AAC — useful when targeting a specific format or reducing file size from lossless sources.

              

              
            

          

        


        
        

          

            
            
            
Whether to include subtitle tracks in the encoded output. Copy passes subtitle tracks through, filtered by your language selection in the Audio tab. Strip removes all subtitles. Note: subtitles are always dropped for MP4 and WebM containers regardless of this setting.

          

        


        
        

          

            
            
            
Files with a bitrate above this value are flagged for optimization. Default: 25000 kbps (25 Mbps). Files flagged by bitrate are good candidates for re-encoding since modern codecs can typically match or exceed their quality at a fraction of the size. Takes effect on future scans — rescan your folders after changing this to re-evaluate existing files.

          

          

            

              

                Flag AV1 files for optimization
                
Flag AV1 files so they appear in the optimization queue. Disable this if AV1 is your preferred output codec or your devices support AV1 natively. Takes effect immediately — no rescan needed.

              

              
            

          

        


        
        

          

            

              

                Enable Web UI
                
Allow other devices on your network to access the Conduit interface via a web browser. When disabled, Conduit only listens on localhost. Requires a full app restart to take effect.

              

              
            

          

          

            

              
              
              
IP address to listen on. Use 0.0.0.0 to accept connections on all network interfaces, or enter a specific IP (e.g. 192.168.1.10) to restrict access to one interface. Requires a restart to take effect.

            

            

              
              
              
Port that network clients connect to. Default: 8000. Make sure this port is not blocked by your firewall. Requires a restart to take effect.

            

            

            

              
              

                
                
              

              
Set a username and password to protect the Web UI with HTTP Basic Auth. Leave both blank for no authentication. Credentials are stored in config.json. Requires a restart to take effect.

Presets let you save and quickly load a full set of encode settings in the Encode modal. Useful for workflows you repeat often — for example, a Tower Unite preset, a high-quality archive preset, or a quick 720p web preset. Built-in presets are read-only.
